# WhiteSpaceRemoval
 Remove white spaces!
 
 This app uses shortcuts:  
 F1 - Remove white spaces from input  
 F2 - Copy to clipboard  
 F3 - Clear both inputs  
 F4 - Clear clipboard  
 F5 - Pin (App always on top / App can be minimized)  

Green/Red indicator tells if the app is focused or not.
